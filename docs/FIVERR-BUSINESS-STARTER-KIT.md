## Intake Form (Ask Before Touching Any Tool)

Send this to the buyer immediately after order:

1. What is your business name and what do you do? (one sentence)
2. Do you have a primary color in mind, or are you open to suggestions?
3. Any colors you absolutely do not want?
4. Name 2–3 brands whose look and feel you like. (Not copying — just for vibe.)
5. Preferred feel: clean/minimal · bold/confident · warm/friendly · luxe/premium
6. Full contact info: name, title, phone number, website, which social handles
7. Do you have a logo file to include on the brand board? (If no, note that logo is not included in this gig.)

---

## Approval Workflow (Reduces Revisions and Refunds)

1. Receive intake form answers
2. Run Palette Studio → send buyer 3 palette options → **get written approval before proceeding**
3. Build everything else from approved palette
4. Package ZIP → deliver via Fiverr Deliver Work button
5. Write a clear delivery message explaining what is in each folder

Palette approval at step 2 eliminates most downstream color complaints. Everything else flows from that committed choice.

## New Tools to Build

### 1. Brand Board Generator
**Priority: High — needed for every delivery**

A single-page PDF generator. Input: hex codes (paste from Palette Studio), font names, logo upload, business name, tagline. Output: clean one-page brand reference PDF.

- Pulls color swatches with hex codes
- Shows heading font large, body font small, both named
- Includes logo placement (upload field)
- Exports as PDF
- Opsette-branded footer

### 2. ZIP Builder / Brand Kit Assembler
**Priority: High — needed for scale**

A tool that accepts file uploads (PDFs, PNGs, HTML files) from all the other tools, lets you organize them into named folders, and outputs a single branded ZIP file.

- Drag-and-drop file input per folder slot
- Pre-labeled folder structure matching the kit
- Renames files cleanly on output
- Adds Opsette branding to ZIP name
- One-click download

### 3. Font Pairing Tool
**Priority: Medium — completes the brand kit story**

Either a standalone tool or an extension of Palette Studio. User picks a vibe (modern, warm, bold, minimal) and gets a Google Fonts heading + body font pairing with a live preview.

- Vibe-based selection (not manual font browsing)
- Live preview of heading + body together
- Outputs font names for use in Brand Board Generator
- Exports a simple Font Pairing PDF

### 4. Social Banner Builder
**Priority: Medium — premium tier add-on**

Extension of Icon Kit. User uploads logo, picks brand color, adds a short tagline. Outputs banner images sized for LinkedIn, Facebook, and Twitter/X.

- Canvas layout with logo + color background + tagline
- Export at correct dimensions per platform
- Could live inside Icon Kit as a new tab/mode
